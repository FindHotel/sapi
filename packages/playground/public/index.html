You can run search manually in the console:

const search = await Sapi.client.search({
  checkIn: "2021-01-08",
  checkOut:"2021-01-10",
  rooms:"2",
  rates: true
})

const results = search.getHitsWithRates()

const nextPage = await search.loadMore()
